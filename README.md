# angular-confirm-page
Angular app with confirm page on start

## Test app 
[angular-confirm-page](https://sebastiangolian.github.io/angular-confirm-page)

## Getting started
```bash
npm install
```
```bash
npm run start
```

## Scripts
Build project
```bash
npm run build
```
Test project
```bash
npm run test
```
Test e2e project
```bash
npm run e2e
```
Lint project
```bash
npm run lint
```
Deploy files in gh-pages
```bash
npm run deploy
```
